Start
Lemonade Change (Easy) – Simple “do we have change?” greedy. Good warm-up.
Jump Game (Medium) – “Max reach” as you scan. Very common and reusable pattern.
Maximum Subarray (Medium) – Kadane’s algorithm: one pass, track current sum and best sum. Another classic.

Next
Jump Game II – Same theme as Jump Game, with a “min jumps” twist.
Partition Labels – Intervals / “last index of each letter” style greedy.
Gas Station – “Can we complete a circular route?” – different but standard.

Later
Hand of Straights, Merge Triplets, Valid Parenthesis String – more variation.
Candy (Hard) – two passes (left–right, right–left); do after you’re comfortable with the others.
